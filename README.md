# DermaCure AI – Skin Disease Prediction Using Deep Learning

## Overview

DermaCure AI is an AI-powered skin disease prediction system that leverages Deep Learning and Computer Vision techniques to identify skin diseases from medical images. The project uses Convolutional Neural Networks (CNNs) built with TensorFlow and Keras to analyze skin images, extract meaningful features, and predict potential skin conditions.

The primary goal of this project is to demonstrate the application of Artificial Intelligence in healthcare by developing an intelligent diagnostic support system for early skin disease detection.

In addition to AI-based prediction, the platform includes a doctor review workflow where patients can share AI-generated reports with doctors for further analysis and recommendations.

## How It Works

1. User uploads a skin image.
2. The image is preprocessed and normalized.
3. The trained CNN model extracts image features.
4. TensorFlow predicts the most likely skin disease.
5. Results are displayed through the web interface.
6. The generated report can be shared with a doctor.
7. The doctor reviews the report and provides recommendations.
8. The reviewed report is returned to the patient.
